Quick Start

5 分钟完成迁移

改一行 Base URL，其他照搬。兼容 OpenAI 官方 Python / Node.js / cURL 调用方式。

demo.py 
from openai import OpenAI

client = OpenAI(
    base_url="https://cy5u.com/v1",   # 只需改这一行
    api_key="sk-your-token5u-key",
)

resp = client.chat.completions.create(
    model="gpt-5.4",
    messages=[{"role": "user", "content": "你好，词元无忧"}],
)
print(resp.choices[0].message.content)
import OpenAI from "openai";

const client = new OpenAI({
  baseURL: "https://cy5u.com/v1",
  apiKey: "sk-your-token5u-key",
});

const resp = await client.chat.completions.create({
  model: "claude-opus-4-7",
  messages: [{ role: "user", content: "你好，词元无忧" }],
});
console.log(resp.choices[0].message.content);
curl https://cy5u.com/v1/chat/completions \
  -H "Authorization: Bearer sk-your-token5u-key" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gemini-3.1-pro",
    "messages": [{"role": "user", "content": "你好，词元无忧"}]
  }'
from anthropic import Anthropic

client = Anthropic(
    base_url="https://cy5u.com",
    api_key="sk-your-token5u-key",
)

msg = client.messages.create(
    model="claude-opus-4-7",
    max_tokens=1024,
    messages=[{"role": "user", "content": "你好"}],
)
print(msg.content[0].text)

常见问题

从 OpenAI / Anthropic 官方迁移到词元无忧，要改多少代码？
绝大多数情况只需要改两处：Base URL 改成 https://cy5u.com（或 https://cy5u.com/v1），以及把 API Key 换成在词元无忧控制台生成的 Key。原先 OpenAI Python / Node.js SDK、Claude SDK、LangChain 等调用方式全部保留，无需重写业务逻辑。
价格是如何算的？有没有订阅费或最低消费？
完全按实际消耗的 Token 量计费，没有订阅、没有月租、没有最低消费；主流模型在官方定价基础上提供 5 折起的倍率优惠。账单支持按日 / 按接口 / 按 Key 维度查询。
稳定性怎么保障？万一某家供应商出现故障怎么办？
同一模型配置多渠道并行，遇到上游限流或 5xx 会自动切换下一条可用线路，对业务端几乎无感。全年可用性承诺 99.9%，后台状态看板实时公开，重要故障会通过公告 / 邮件通知。
公司使用能开正规发票吗？
能，如需开票请联系客服
支持哪些客户端 / 应用直接导入？
支持 Cherry Studio、Lobe Chat、OpenCat、AI Workspace、AionUI、AMA 问天、流畅阅读、CC Switch 等主流应用的一键密钥导入，控制台提供专属协议链接，复制 → 打开客户端即完成绑定。
